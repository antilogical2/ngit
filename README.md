# ngit
ngit is git for beginners. very simple
# setup
```
git clone https://github.com/eshnd/ngit ~/.ngit
cd ~/.ngit
make
```
# usage
clone repos normally:        
```
git clone https://github.com/username/repo
```       
ngit to update:
```
ngit commit_message
```
instead of
```
git add --all
git commit -m "commit_message"
git push
```
